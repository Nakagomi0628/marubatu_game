逆転OXゲーム
スキルを発動したりステージのギミックを利用して逆転を狙える新感覚のOXゲームです

遊び方
1. pdeファイルをprocessingで開いて実行します
 - ダウンロードはhttps://processing.org/
2. startボタンを押します
3. PlayerAボタンを押します
 - 紫色のボタンでスキルを選びます
 - Resetボタンでスキルをリセットします
 - Backボタンで元の画面に戻ります
4. PlayerBボタンを押します
 - 紫色のボタンでスキルを選びます
 - Resetボタンでスキルをリセットします
 - Backボタンで元の画面に戻ります
5. Stageボタンを押します
 - 黄色のボタンからステージを選びます
 - Backボタンで元の画面に戻ります
6. OKボタンを押します
 - ステージを押して交互に印を書きます
 - 印を書くとspが溜まります
 - spがコストを超えるとスキルを発動できます
 - 印を5つ揃えると元の画面に戻ります

スキル
1. スキルはサイズ・コスト・効果から構成されます
2. サイズの合計が4になるようにスキルを選びます

ステージ
1. ステージは固有の形とギミックを持っています
2. 印を書くとギミックが発動します
